DEKLARASI:
N, i: Number

--------------------------------------------
DESKRIPSI: 

START
INPUT N
SET i <- 1

FOR i to N
   IF i Mod 3 is 0 THEN
	DISPLAY "Fizz"
   ELSE IF i mod 5 is 0 THEN
	DISPLAY "Buzz"
   ELSE IF i mod 5 is 0 AND i mod 3 is 0 THEN
	DISPLAY "FizzBuzz"
   ELSE 
	DISPLAY i
   ENDIF
   ENDIF
   ENDIF	
ENDFOR 
STOP
